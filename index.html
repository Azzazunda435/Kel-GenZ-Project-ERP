Bill of Material (BOM)

Perhitungan biaya produksi dengan struktur hierarkis komponen

1. Konsep dan Rumus Dasar BOM

Bill of Material (BOM) adalah daftar seluruh komponen yang dibutuhkan untuk membuat satu produk, lengkap dengan jumlah dan biaya.

Rumus umum biaya BOM:

Biaya Komponen = Jumlah Komponen × Harga Satuan

Total Biaya Produk = Σ (Biaya seluruh komponen)

Forecasting - Trend Linear

Prediksi permintaan menggunakan metode Least Square

1. Konsep dan Rumus Dasar Forecasting Trend Linear

Forecasting Trend Linear menggunakan metode kuadrat terkecil (least square) untuk memprediksi nilai di masa depan berdasarkan data historis.

Persamaan Trend Linear:

Y = a + bX

Dimana:

  • Y = Nilai prediksi
  • X = Periode waktu
  • a = Intersep (konstanta)
  • b = Slope (koefisien kemiringan)

Rumus perhitungan:

b = (n∑XY - ∑X∑Y) / (n∑X² - (∑X)²)

a = (∑Y - b∑X) / n

Job Sequencing Method (JSM)

Penjadwalan optimal untuk 2 mesin dengan aturan Johnson

1. Konsep dan Rumus Dasar JSM

Job Sequencing Method menggunakan algoritma Johnson untuk menentukan urutan optimal pekerjaan pada dua mesin untuk meminimalkan makespan (total waktu penyelesaian).

Aturan Johnson:

  1. Cari waktu proses terkecil dari semua job di kedua mesin
  2. Jika waktu terkecil di Mesin 1, tempatkan job di awal urutan
  3. Jika waktu terkecil di Mesin 2, tempatkan job di akhir urutan
  4. Hapus job yang sudah dijadwalkan dan ulangi hingga semua job terjadwal

Perhitungan:

  • Makespan = Total waktu penyelesaian semua job
  • Idle Time = Waktu menganggur mesin

Input Data Pekerjaan

Waktu Proses Job pada Mesin 1 dan Mesin 2

Simple Additive Weighting (SAW)

Metode pengambilan keputusan multi-kriteria dengan pembobotan

1. Konsep dan Rumus Dasar SAW

SAW adalah metode MCDM (Multi-Criteria Decision Making) yang menghitung skor preferensi alternatif berdasarkan normalisasi nilai dan bobot kriteria.

Normalisasi:

Benefit: rij = xij / max(xij)

Cost: rij = min(xij) / xij

Skor Preferensi:

Vi = Σ (wj × rij)

Dimana:

  • Vi = Skor preferensi alternatif ke-i
  • wj = Bobot kriteria ke-j
  • rij = Nilai normalisasi

Market Basket Analysis

Analisis asosiasi untuk menemukan pola pembelian produk

1. Konsep dan Rumus Dasar Market Basket

Market Basket Analysis menggunakan aturan asosiasi untuk menemukan pola "jika membeli X maka membeli Y".

Metrik Utama:

Support: Frekuensi kemunculan itemset dalam transaksi

Support(A) = Transaksi mengandung A / Total Transaksi

Support(A,B) = Transaksi mengandung A dan B / Total Transaksi

Confidence: Probabilitas membeli B jika membeli A

Confidence(A→B) = Support(A,B) / Support(A)

Lift Ratio: Kekuatan asosiasi

Lift(A→B) = Confidence(A→B) / Support(B)

  • Lift > 1: Asosiasi positif (saling mendorong)
  • Lift = 1: Independen
  • Lift < 1: Asosiasi negatif

Profile Matching

Evaluasi kandidat dengan analisis gap terhadap profil ideal

1. Konsep dan Rumus Dasar Profile Matching

Profile Matching membandingkan profil kandidat dengan profil ideal menggunakan Gap Analysis dan pembobotan Core Factor & Secondary Factor.

Tahapan Perhitungan:

  1. GAP: GAP = Nilai Kandidat - Nilai Ideal
  2. Bobot Nilai: Konversi GAP ke bobot berdasarkan tabel
    SelisihBobotKeterangan
    05Kompetensi sesuai
    14.5Kompetensi lebih 1
    -14Kompetensi kurang 1
    23.5Kompetensi lebih 2
    -23Kompetensi kurang 2
    32.5Kompetensi lebih 3
    -32Kompetensi kurang 3
    ≥4 atau ≤-41Kompetensi sangat tidak sesuai
  3. Core Factor (CF): NCF = Σ Bobot CF / Σ Item CF
  4. Secondary Factor (SF): NSF = Σ Bobot SF / Σ Item SF
  5. Nilai Total: N = 60% × NCF + 40% × NSF
  6. Ranking: Urutkan berdasarkan Nilai Total tertinggi

Markov Chain

Analisis probabilitas transisi antar state untuk prediksi masa depan

1. Konsep dan Rumus Dasar Markov Chain

Markov Chain adalah model stokastik yang menggambarkan transisi antar state dengan probabilitas tertentu, dimana state masa depan hanya bergantung pada state saat ini.

Konsep Utama:

  • State: Kondisi atau situasi yang mungkin terjadi
  • Transition Probability (Pij): Probabilitas berpindah dari state i ke state j
  • Transition Matrix (P): Matriks berisi semua probabilitas transisi

Properti:

  • Σ Pij = 1 untuk setiap baris (total probabilitas keluar = 1)
  • 0 ≤ Pij ≤ 1 (probabilitas antara 0 dan 1)

Prediksi State Masa Depan:

π(n) = π(0) × P^n

Dimana:

  • π(n) = Distribusi probabilitas pada periode n
  • π(0) = Distribusi probabilitas awal
  • P^n = Matriks transisi pangkat n

Steady State: Kondisi keseimbangan jangka panjang dimana π = π × P
